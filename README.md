# GPT-2 Text Generation 🤖

## What is this project?
This project fine-tunes GPT-2 on a custom dataset of 
stories and jokes to generate creative and coherent text.

## How it works?
1. Load GPT-2 pre-trained model
2. Train it on custom stories and jokes dataset
3. Generate new creative text from any prompt

## Sample Outputs

Prompt: "Once upon a time"
Output: Once upon a time in the forest, there lived an old 
man named Leo... for their journey home three generations 
to come new learnings...

Prompt: "Why did the"
Output: Why did the tomato turn red? Because it saw a 
picture of his favorite book!

Prompt: "There was a girl named"
Output: There was a girl named Mia who lived alone in an 
apartment filled with dreams. She never felt lonely again!
